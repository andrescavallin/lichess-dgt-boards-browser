How it works

This program will communicate with Lichess using the Board API and with DGT Board using LiveChess 2.X
If you already configured your API Access Token click Play below

Lichess Requirements

In order to use this program you will need to create an account on Lihcess.org
If you don't an account yet, please register here on this link: Lichess Registration
You also need a Lichess API Access Token.
To get an access token please visit this link: Get API Access Token
Lastly you need to configure the API Access Token on the configuration section.
If you have not done so already, visit the Configuration Section

DGT Board Requirements

To connect to the DGT Electronic Board you will need to install DGT LiveChess 2.2.5 or later
You can download the software here: LiveChess 2.X
This page and LiveChess must be on the same local network.
If LiveChess is running on this same computer, you can check you connection to LiveChess by opening this link: DGT LiveChess Documentation
If LiveChess is running on a different machine or different port,
you will need to set the IP address and port here on the Configuration Section
